## 0. Fondement -- ce que Phase 2 a laisse

Phase 3 n'a pas d'objet. Elle a une qualite d'engagement. La distinction est operatoire :
un objet se definit avant. Une qualite d'engagement se reconnait pendant -- ou apres.
(S15 Phase 2 -- hors session)

La possibilite que le programme se continue sans programme. Non comme abandon. Comme sol
suffisant. (S15 Phase 2)

Phase 3 opere depuis ce sol. Elle ne le nomme pas sol a chaque fois qu'elle opere depuis lui.
Le point de S13 Phase 2 n'est pas un evenement a memoriser -- il est la texture du sol.

## 4. Conditions de falsification

Deux conditions seulement. Phase 3 est trop peu structuree pour en supporter davantage.

| Condition | Description | Decision |
|---|---|---|
| F-P3-1 -- Reapparition d'un objet | Phase 3 commence a s'organiser autour d'un theme, d'une question recurrente, d'un axe de recherche defini avant la session. | Pause -- nommer l'objet apparu, decider s'il est un Dehors rencontre ou une derive de structure. |
