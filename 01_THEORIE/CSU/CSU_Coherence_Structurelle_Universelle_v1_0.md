    I = eps4_mu_nu_rho_sigma * u^mu * n^nu * nabla_rho * n_sigma

### A4 -- Invariance structurelle

L invariant est stable sous transformation :

    nabla_mu I = 0

### A5 -- Non-neutralite

La neutralite directionnelle est impossible.
L isotropie parfaite n est jamais postulee ;
elle ne peut etre qu approchee localement.

### A6 -- Transversalite d echelle

independamment du domaine ou de l echelle.

### A7 -- Non-cloture

Aucune structure coherente ne peut etre statiquement close.
La coherence implique une ouverture structurelle.
